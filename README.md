# ✋ Gesture Detection System

A real-time hand gesture detection system built using **Python**, **OpenCV**, and **Machine Learning**.

## 🚀 Features

* Real-time hand tracking using webcam
* Gesture recognition using trained model
* Image preprocessing and classification
* Clean modular code structure

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy
* TensorFlow / Keras

## 📂 Project Structure

```
gesture-detection/
│── Data/           # Dataset (if included)
│── Model/          # Trained model files
│── Requirement/    # Requirements file
│── src/            # Main source code
│   ├── main.py
│   ├── test.py
│── .gitignore
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/mohitkarki/gesture-detection.git
cd gesture-detection
```

2. Install dependencies:

```
pip install -r Requirement/MatReq.txt
```

3. Run the project:

```
python src/main.py
```
